A While Loop ... To List The Players

An intern at Tottenham Hotspurs soccer club created a 2D array, populated from a database, to store next season's players. We don't have access to their database, so we have simulated the population of the 2D array by plugging in a set of players, in a seed method. Your task? Create code that uses a nested while loop to list player name, age and position. Mouse over the solution card to check your code, only if you need to!

Row Major Order

Start at row 0, column 0. Progess to last row and last column. The number of rows is the length of the 2D array. The number of columns is the length of any row.

The Strategy

Listing the players requires traversal of the 2D array, iterating through the rows, and for each row, iterating through the columns. A nested loop facilitates this nicely! Try it for yourself first, mouse over the card to the right, to compare your solution. Remember that each element of the 2D array is accessed by the row and the column. In this case, each element is also an object, instantiated from class, SoccerPlayer. Here is the pseudo-code ... 


call the seeding method, initialize 2D array

while ... check number of rows
    while ... check number of columns
        print player name, age, position
        increase inner loop variable
    increase outer loop variable

// Traversing a 2D array, row-major order, 
// and a while loop ... and soccer players    
public static void traverseRMOWhileLoop() {

 SoccerPlayer[][] players = seedPlayers();
 int row = 0;
 while (row < players.length) {
   int col = 0;
   while (col < players[0].length) {
     SoccerPlayer player = players[row][col];
     System.out.println(player.getName() + " is " + 
                        player.getAge() + 
                        " years old and plays as a " + 
                        player.getPosition());
    col++;
    }
   row++;
   }
}
